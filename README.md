# Keyboard Events

## Main Page:

's' --Start/Stop the timer

'h' --open the help pop-up

'r' --open the reset pop-up

';' --open the setting pop-up

'a' --open the add task pop-up

## In Pop-Ups

'Enter' --confirm your setting

'Escape' --cancel the change and close the pop-up
